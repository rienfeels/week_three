Display Properties (this is an h3)

  • Inline: Default behavior for elements (it's usually reset to "block" by browsers*).
    • Doesn't break the flow, stays inline
    • Margin and Padding can be applied
    • Height and Width are ignored
  • Inline-Block: Similar to inline but it respects width and height values.
  • Block: Are not inline, more like "stacked." If a width isn't set, they'll take up the full width by default
